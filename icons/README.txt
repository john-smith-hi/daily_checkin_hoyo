Icon Files Needed
=================

Please add the following icon files to this directory:

1. Extension Icons:
   - icon16.png (16x16 pixels) - Small extension icon
   - icon48.png (48x48 pixels) - Medium extension icon  
   - icon128.png (128x128 pixels) - Large extension Icon

2. Check-in Item Icons:
   - checkin1.png (64x64 pixels) - First check-in item icon
   - checkin2.png (64x64 pixels) - Second check-in item icon
   - checkin3.png (64x64 pixels) - Third check-in item icon
   - checkin4.png (64x64 pixels) - Fourth check-in item icon

The extension will work without these files, but the images won't display properly.
